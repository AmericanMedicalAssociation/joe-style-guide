### Description
The current issue teaser appears in the issue listing page on first load. It displays the current issue with a description and 3 articles from the issue.


### Variables
~~~
date:
  type: string / required
title: {
  text :
    type: string / required
  href :
    type: string / required
}
description:
  type: string / optional
number:
  type: string / optional
~~~
