### Description
The article footer appears at the end of an article. It contains reference information.

### Variables
~~~
{
references: see references molecule
editor_note:
  type: string / optional
citation:
  type: string / required
doi:
  type: string / required
acknowledgements:
  type: string / optional
disclosure:
  type: string / optional
ethics_case:
  type: boolean / required
}
~~~
