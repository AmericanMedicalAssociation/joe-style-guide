### Description
This is the extended teaser pattern

### Variant options
* This teaser has color variants. To use it set a variant style to "reverse".

### Variables
~~~
teaser: {
  flag: 
    see flag atom
  authors:
    type: array / optional 
  title: 
    type: string / required
  url:
    type: string / required
  image: 
    type: media / optional
  size: 
    type: string / optional
  style:
    type: string / optional
  citation:
    type: string / required
  doi:
    type: string / required
}
~~~
