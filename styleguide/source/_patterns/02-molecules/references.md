### Description
This Pattern shows an articles references. They are displayed in an ordered list and can contain up to three reference links to google scholar, pubmed, or crossref.

### Variables
~~~
references {
  number:
    type: string / required
  text:
    type: string / required
  google_scholar:
    type: url / optional
  pubmed:
    type: url / optional
  crossref:
    type: url / optional
}

~~~
