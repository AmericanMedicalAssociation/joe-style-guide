### Description
This is the standard input pattern.

### Variables
~~~
input: {
  type:
    type: string (url) / required
  name: 
    type: string / required
  id: 
    type: string / required
  value:
    type: string / optional (ex: "submit", "reset")
  placeholder:
    type: string / optional ("secondary" or "")
}
~~~
