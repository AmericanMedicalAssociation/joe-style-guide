### Description
The drawer trigger is used to control drawers. It has an optional icon that uses grunticon to render.

~~~
trigger {
  id: 
    type: string / required
  class: 
    type: string / optional
  control_id: 
    type: string / required
  text: 
    type: string / required
  icon.symbol:
    type: string / optional
  icon.text:
    type: string / optional
}
~~~
