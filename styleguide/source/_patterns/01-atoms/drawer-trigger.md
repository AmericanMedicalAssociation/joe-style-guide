### Description
The drawer trigger is used to control drawers.

~~~
trigger {
  id: 
    type: string / required
  class: 
    type: string / optional
  control_id: 
    type: string / required
  text: 
    type: string / required
  visible:
    type: boolean / required (controls text label visibility)
  icon:
    type: svg / optional
}
~~~
